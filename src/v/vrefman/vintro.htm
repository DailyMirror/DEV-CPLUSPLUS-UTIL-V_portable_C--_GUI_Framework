The V C++ GUI Framework
 
User Guide
 
by
Bruce E. Wampler, Ph.D.
 



What is V? This is a short introduction to V, why it was written, and what it can do for you.
Tutorial The V Tutorial is a tutorial introduction to V. It has a general introduction to GUI concepts, and then covers the basics of developing an application using V.
V Topics This section includes various V topics, including Release Notes, the FAQ, quick reference, and guides for the application programs included with the V distribution.
Installing V This section covers installation of V on the various platforms using different compilers. It is where to start if you are just installing V.
V Reference The V Reference Guide is intended as a one stop source of information for the experienced V programmer. It is an alphabetical reference to all V classes, values, utilities, and application programs. Use it with this full V User Guide by clicking V Reference, or as a stand alone reference without the top navigation bar by opening vrefman.htm here, or directly with your browser.



This is the V User Guide. Beginning with V 1.21, the reference manual is available only as an HTML document, and includes all the material formerly included in the LaTeX V Reference Manual.

The organization of the User Guide is based on HTML frames. The top frame is the main navigation aid, and allows you to get to any section of this guide at any time. If you are just getting started, begin with What is V?. Then read the Tutorial.
